tofucode

A web UI for Claude Code

tofucode is a browser-based interface for Claude Code — Anthropic's AI coding agent. It lets you run, manage, and interact with Claude on any machine — local or remote — entirely from a browser tab.

Built for developers who work across environments, tofucode wraps Claude Code in a clean, responsive UI with real-time streaming, multi-session support, a file browser, terminal access, and plan mode — so you can review what Claude intends to do before it does it.

Install it globally, run it on a VM, or pull the Docker image. Once it's running, your AI coding agent is just a browser tab away.
